# Output Formats

Use the following output structures.

## 1. Candidate shortlist

```text
Here are the best current AI/tech items to consider for X:

1. [Topic]
Source:
Published:
What happened:
Why it matters:
Why it may perform on X:
Suggested angle:
Priority: A/B/C

2. [Topic]
...

Reply with the numbers you want, and I will turn them into post templates.
```

## 2. Post generation

```text
Topic:
Core angle:

Version A - Quick Take
[post]

Version B - 3-Point Breakdown
[post]

Version C - Discussion Hook
[post]
```

## 3. Optional reply starters

If useful, add:

```text
Reply ideas:
1.
2.
3.
```

## Writing standards

- Keep the first sentence punchy.
- Prefer concrete nouns over abstract hype.
- Avoid empty lines unless the platform style benefits from them.
- Use short paragraphs for Chinese X posts.
- If the topic is technical, translate it into user impact.
- If the topic is crowded, choose a sharper angle instead of repeating the headline.
